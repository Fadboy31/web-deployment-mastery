# Web Deployment Mastery

## Quest 2: Initialize Git Repository

**Mission:** Turn your project folder into a Git repository so all your work is tracked and versioned.

### Steps Completed:
- Initialized Git repository (`git init`)  
- Checked status (`git status`)  
- Created `.gitignore` file  
- Added all files (`git add .`)  
- Made first commit (`git commit -m "Initial commit"`)  

✅ Project is now tracked with Git and ready for deployment.
