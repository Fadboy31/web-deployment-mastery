# QUEST 5: Deploy to GitHub Pages

## Mission
Deploy your website for FREE on GitHub Pages and make it live on the internet.

---

## What is GitHub Pages?
GitHub Pages is **free static website hosting** directly from your GitHub repository. Once set up, any changes you push to your repository automatically update your site.

---

## Deployment Steps

### 1️⃣ Go to Your Repository
- Open your repository on GitHub: [https://github.com/Fadboy31/web-deployment-mastery](https://github.com/Fadboy31/web-deployment-mastery)

### 2️⃣ Access Pages Settings
- Click **Settings → Pages** (in the left sidebar).

### 3️⃣ Select Branch
- Under **Source**, select **main branch**.  
- Click **Save**.
